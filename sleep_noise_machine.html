Modal noise machine with harmonic evanescence

Offline browser synthesizer for slow drones, filtered noise, modal note drift, subtle entrainment tones, fading harmonics, echo, and dark reverb. Press Start audio first; browser audio starts only after a user gesture.

Current patch is intentionally bounded: capped master gain, capped delay feedback, capped resonance, slow envelopes, dark filtering, and smoothed changes on active audio parameters. Binaural, monaural, and isochronic modes are implemented as sound textures, not medical sleep treatment claims. Binaural mode needs headphones to produce separate left/right tones.
Single-file Web Audio build. No external libraries, no network dependency, no sample files.
